# Jogo da Forca

## Descrição
Este é um projeto em C que implementa o clássico jogo da forca.  
O objetivo do jogador é adivinhar a palavra correta antes de completar o bonequinho da forca.

---

## Estrutura do Projeto
```
jogo_da_forca
│
├── main.c # Arquivo principal que executa o jogo
├── funcoes.c # Implementação das funções do jogo
├── funcoes.h # Declarações das funções
└── README.md # Este arquivo de documentação
```
---

## Como Compilar e Executar
Para compilar no terminal:
```bash
gcc main.c funcoes.c -o jogo_da_forca
```
Para executar:
```bash
.\jogo_da_forca.exe
```
